На выходе получается таблица с track_id идентификатором песни и count количеством её прослушиваний
Для запуска с облачной ВМ code.ipynb
Для локального запуска code_.py; выполнит все перемещения и преобразования в контейнер

Процесс конфига для 1 ноды и для кластера из 3 нод полностью описан со скриншотами
