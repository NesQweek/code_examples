ТЗ: сделать алгоритм для входа/выхода из позиции на всём временном окне после наступления алерта, вывести полученную прибыль алгоритмом
На скриншоте теханализ на основе данных всего датасета
На видео алгоритм вместо использользования всего датасета основывается только на прошедших свечах, 
известных линиях поддержки/сопротивления и набора индикаторов
ML тут не использовал из-за малого объема данных (~ 3 часовой интервал)
