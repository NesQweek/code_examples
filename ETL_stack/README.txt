Небольшой ETL стек для <16 гб оперативки
Прогнозирует одобрение/отказ ипотечного займа для клиента на основе представленной о нём информации. Датасет взят с Kaggle. 
Cимулирует поступление данных различного формата (avro, orc, parquet...) из различных источников на quorum кластер с Apache Kafka брокерами в соответствующие топики для каждого типа данных. По расписанию проверяет наличие данных в топиках, проверяет их на соответствие схемы в Schema Registry, передает оркестру ML моделей, которые отправляют обобщенный предикт о заемщике в веб приложение.
